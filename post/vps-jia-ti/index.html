VPS架梯

· 2014-02-26 · # 折腾

一直都想搞个vps玩玩，觉得在终端窗口里敲出一个个指令符的样子超酷。之前虽然有过特价的vps，但总是觉得自己连Linux基础都不懂，买了vps也是浪费。这次看到bandwagon在搞特价vps，最低的只要3.99美刀一年，还有4.99、5.99美刀的，虽然配置低了点，但拿来玩玩还不错，纠结了会，入手了5.99刀，配置是内存128M_硬盘3G_流量300G。

下单后，紧急Google了些vps的基础知识，配置了Debian6.0系统，然后用如下命令更改了登录端口和登录密码：

  1. vi _etc_ssh/sshd_config 找到 Port 22，更改数字
  2. passwd 用户名

接着就准备搭建VPN了，基于尽快使用不折腾的原则，打算就搞个简单的pptp，找了几个教程，发现这个最方便有效，为免链接实效，拷贝代码如下：

  1. wget -c http://dl.dropbox.com/u/32817960/linux/Software/pptpd_debian.sh
  2. chmod +x pptpd_debian.sh
  3. ./pptpd_debian.sh_

默认用户名 vpn 密码 123，可在_etc_ppp/chap-secrets 重新配置用户信息。

vpn配置完成后，发现占用内存才20M出头，想想再搭个shadowsocks吧，在网上搜了比较下，最后选择shadowsocks-libev版，但是根据官方的教程总是提示头文件下载失败，于是重新Google了教程，找到这篇，按此教程搭建成功。同样，拷贝代码如下：

  1. apt-get install build-essential autoconf libtool libssl-dev gcc -y
  2. apt-get install git -y
  3. git clone https://github.com/madeye/shadowsocks-libev.git
  4. cd shadowsocks-libev
  5. ./configure
  6. make && make install
  7. nohup _usr_local_bin_ss-server -s IP地址 -p 端口 -k 密码 -m 加密方式 & (例：nohup _usr_local_bin_ss-server -s 156.132.67.213 -p 8981 -k admin888 -m aes-256-cfb &)
  8. vi编辑 _etc_rc.local ，在首行加入 nohup _usr_local_bin_ss-server -s IP地址 -p 端口 -k 密码 -m 加密方式 & (开机自启动)

最后，可用ps -ef命令查看以上服务是否在进程中。shadowsocks客户端选用了Goagent X，还不错，可惜pac文件中似乎未对Google搜索作规则，默认的Google总是进入HK站，只能用 google.com/ncr 来强制进入US站。

PS:这家的vps速度并不是很稳定，感觉一会快一会慢，不过凑合用用还可以了，毕竟这价格，还要啥自行车啊。
